# StillLoud Website

Official website for **StillLoud**, a creative branding and digital design studio helping startups and small businesses build memorable brands through strategic design, websites, and digital experiences.

## 🌐 About

StillLoud specializes in:

- Brand Identity
- Logo Design
- Website Design & Development
- UI/UX Design
- Social Media Design
- Marketing Creatives
- Motion Graphics
- Creative Strategy

---

## 🚀 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- SVG Assets
- Responsive Design
- Vercel (Deployment)


## 📱 Responsive Design

The website is optimized for:

- Desktop
- Laptop
- Tablet
- Mobile

On mobile devices the navigation automatically switches from:

```
fullLogo.svg
```

to

```
logo.svg
```

for a cleaner interface.
